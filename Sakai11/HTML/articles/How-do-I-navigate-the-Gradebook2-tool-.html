To set-up Gradebook, click Edit / Gradebook Settings

Note: OR Right-Click (CTRL-Click MAC) in the word "Gradebook" and select Edit this Gradebook.

To Add Categories, click File / New Category.

Note: OR Right-Click (CTRL-Click MAC) in the word "Gradebook" and select Add New Category.

To add a new item, click File / New Item

Note: OR Right-Click (CTRL-Click MAC) the name of a Category and select Add New Grade Item.
